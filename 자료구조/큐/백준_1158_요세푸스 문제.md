# 요세푸스 문제



## 큐로 풀기

```java
import java.io.*;
import java.util.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringTokenizer st = new StringTokenizer(br.readLine());
        StringBuilder sb = new StringBuilder();
        Queue<Integer> queue = new LinkedList<>();

        int n = Integer.parseInt(st.nextToken());
        int k = Integer.parseInt(st.nextToken());

        for (int i = 1; i < n + 1; i++) {
            queue.add(i);
        }

        int[] ans = new int[n];

        int idx = 0;
        int cnt = 0;
        while (!queue.isEmpty()) {
            int x = queue.poll();       
            cnt++;

            if (cnt % k == 0) ans[idx++] = x;
            //큐에서 cnt가 k의 배수가 될때마다 꺼내서배열에 저장
            else queue.add(x);
            // 나머지는 다시 큐에 저장
        }
        sb.append("<");
        for(int i = 0 ; i < ans.length; i++){
            if(i == ans.length-1) sb.append(ans[i]);
            else sb.append(ans[i] + ", ");
        }
        sb.append(">");
        System.out.println(sb);

    }
}
```

## 나머지를 이용한 수학적으로 풀기

* 아이디어를 생각해내기 어렵지만 훨씬 적은 메모리와 시간복잡도로 가능
* ArrayList를 사용.

<pre>
🐳🐳   아이디어

* jump에 해당하는 k의 배수마다 동적배열에서 삭제.
* 삭제할 때마다, 전체 배열의 길이가 줄어들고, 삭제 한 수는 인덱스가 무조건 다음 k의 배수안에서 삭제되기 때문에,
다음 jump는 (jump - 1) + jump 가 된다.

🐳🐳
</pre>


```java
import java.io.*;
import java.util.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringTokenizer st = new StringTokenizer(br.readLine());
        StringBuilder sb = new StringBuilder();

        int n = Integer.parseInt(st.nextToken());
        int k = Integer.parseInt(st.nextToken());

        ArrayList<Integer> list = new ArrayList<>();
        int[] ans = new int[n];

        for(int i = 1; i<= n; i++) list.add(i);

        int idx = 0;
        int jump = 0;
        while(!list.isEmpty()){
            jump = (--jump + k) % list.size();          // 나머지를 이용해 지울 수 구하기
            ans[idx++] = list.get(jump);            // 지우는 수 배열에 저장
            list.remove(jump);                  // 지우기
        }

        sb.append("<");
        for(int i = 0; i< n; i++){
            if(i == n-1) sb.append(ans[i]);
            else sb.append(ans[i] + ", ");
        }
        sb.append(">");
        System.out.println(sb);




    }
}


```
